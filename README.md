# BNB City

**Построй бизнес -- забери x3**

BNB City -- trustless изометрическая игра-город на BNB Smart Chain (BSC). Все игроки строят в одном общем пиксельном городе. Каждое здание генерирует ежедневный доход через дефляционную модель перераспределения с адаптивной ставкой и эпохальным халвингом.

## Почему BNB City?

- **Общий мир** -- все строят в одном городе, видимом каждому
- **Саморегулирующаяся экономика** -- adaptive rate подстраивает доходность под здоровье пула
- **Дефляционная модель** -- 7 эпох снижают вознаграждения со временем (как halving Bitcoin)
- **Полностью on-chain** -- один неизменяемый смарт-контракт, без прокси, без админ-ключей
- **Прозрачные комиссии** -- все сборы видны и верифицируемы на блокчейне
- **8 тиров зданий** -- от 0.001 до 5 BNB, каждый с уникальными модификаторами ставки и капа
- **Бусты и синки** -- расширенная система улучшений для активных игроков

## Быстрый старт

1. Подключи кошелёк (MetaMask, Rabby, Trust Wallet)
2. Купи участок земли у мэра города
3. Построй бизнес на своём участке
4. Расширяй (compound) ежедневно, чтобы увеличить доход
5. Снимай выручку когда готов

## Ключевые цифры

| Параметр | Значение |
|----------|----------|
| Депозитная комиссия | 10% |
| Налог на снятие | 13--16% |
| Базовая дневная ставка | 2--5% (зависит от эпохи) |
| Макс. выплата (cap) | 2.4x--3.0x (зависит от эпохи) |
| Тиры зданий | 8 (от 0.001 до 5 BNB) |
| Типы земли | 4 (от 0.002 до 0.125 BNB) |
| Цикл заработка | 12 часов |
