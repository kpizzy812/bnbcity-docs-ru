# BNB City

**Построй бизнес — забери x3**

BNB City — trustless изометрическая игра-город на BNB Smart Chain (BSC). Все игроки строят в одном общем пиксельном городе. Каждое здание генерирует ежедневный доход через дефляционную модель перераспределения.

## Почему BNB City?

- **Общий мир** — все строят в одном городе, видимом каждому
- **Саморегулирующаяся экономика** — adaptive rate подстраивает доходность под здоровье пула
- **Дефляционная модель** — 7 эпох снижают вознаграждения со временем (как halving Bitcoin)
- **Полностью on-chain** — один неизменяемый смарт-контракт, без прокси, без админ-ключей
- **Прозрачные комиссии** — все сборы видны и верифицируемы на блокчейне

## Быстрый старт

1. Подключи кошелёк (MetaMask, Trust Wallet и др.)
2. Купи участок земли у мэра города
3. Построй бизнес на своём участке
4. Расширяй ежедневно, чтобы увеличить доход
5. Снимай выручку когда готов
